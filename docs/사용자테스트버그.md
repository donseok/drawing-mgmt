## 3. 결함 상세 (BUG-01 ~ BUG-21)

### 🔴 Critical

**BUG-01. DWG/DXF 미리보기 즉시 예외**
- 재현: `/viewer/att-5` → DXF 또는 DWG 탭
- 증상: `TypeError: this.options.clearColor.getHex is not a function`
- 원인 추정: three.js 버전 불일치(`Color` 인스턴스 대신 number 전달)
- 영향: 도면 미리보기 자체 불가 → **핵심 가치 미작동**
- 권장: 뷰어 초기화 시 `new THREE.Color(0xRRGGBB)`로 강제 변환

**BUG-02. 관리자 페이지 RSC 크래시**
- 재현: `/admin` 진입
- 증상: `Attempted to call map() from the server but map is on the client`
- 원인 추정: 서버 컴포넌트에서 클라이언트 전용 함수 직렬화
- 영향: 관리 콘솔 진입 불가
- 권장: 해당 컴포넌트에 `"use client"` 지정 또는 클라이언트 경계 분리

**BUG-03. 삭제 확인 다이얼로그 없음**
- 재현: 행 메뉴 → 삭제
- 증상: 즉시 삭제, 되돌리기 없음
- 영향: 사용자 실수 시 데이터 손실 위험
- 권장: 모달 확인 + Undo 토스트 5초 제공

**BUG-04. 모바일/태블릿 반응형 붕괴**
- 재현: 390×844, 768×1024
- 증상: 본문 영역 비표시, 사이드바가 콘텐츠를 덮음
- 권장: 사이드바 breakpoint 처리, 레이아웃 그리드 재정의

### 🟠 High

**BUG-05. 등록 후 목록 미반영**
- 재현: 신규 등록 → 저장 → 검색 목록 확인
- 증상: 성공 토스트는 뜨지만 항목 미표시
- 권장: 등록 mutation 후 SWR/React-Query `invalidateQueries`

**BUG-06. 수정(Update) UI 부재**
- 재현: 행 메뉴/상세 어디에도 수정 진입점 없음
- 권장: 행 메뉴 + 상세 헤더에 `수정` 버튼 추가

**BUG-07. 결재 `승인/반려` 버튼 무반응**
- 재현: `/approval?box=waiting` → 항목 선택 → 승인
- 권장: 핸들러 바인딩 점검, 처리 결과 토스트 표시

**BUG-08. 빈 결재 상신 가능**
- 재현: `새 결재 상신` 모달에서 모든 필드 공백 → 제출
- 권장: 클라이언트 + 서버 양쪽 검증 추가

**BUG-09. 로비 `검토 회신` 버튼 무반응**
- 재현: 로비 패키지 상세에서 `검토 회신`
- 권장: 라우팅 또는 액션 핸들러 점검

### 🟡 Medium

**BUG-10. `Cmd+K` 단축키 미동작** – 헤더 버튼은 가능. 글로벌 키바인딩 누락.

**BUG-11. 알림 `모두 읽음` 미반영** – 서버 호출 후 클라이언트 상태/배지 업데이트 누락.

**BUG-12. AI 챗봇 입력 비활성** – `<input>`이 readonly 또는 onChange 미연결로 추정.

**BUG-13. 시트셋 보기 토글 미동작** – 토글 버튼이 시각적 상태만 바뀌고 결과 화면 미변경.

**BUG-14. 다중선택 툴바 레이아웃 깨짐** – 버튼이 세로 배치, 라벨 잘림.

**BUG-15. 결재 상세 우측 패널 클리핑** – overflow 처리 부재.

**BUG-16. `[Table] Column with id 'registeredAt' does not exist`** – 일부 화면 콘솔 경고. 데이터 키 불일치.

**BUG-17. `개정` 버튼 무반응** – `/objects/obj-2` 상세.

### 🟢 Low

**BUG-18. 일괄 등록 버튼 무반응** – 홈 카드.

**BUG-19. Tab 포커스 링 일부 누락** – 접근성 저하.

**BUG-20. dev 업로드 페이지 노출** – `/dev/upload` 운영 빌드에서 노출 가능성. 가드 필요.

**BUG-21. 콘솔 다중 경고** – React key 누락, deprecated API 사용 등.
